Skynet Series Artifical Intelligence | Spring 2015, Spring 2016

Chinese Checkers is a bot written in C++ that plays chinese checkers using the minimax algorithm with improvements such as transpostion tables and alpha-beta pruning.

Pathfinder is a single agent search bot written in C++ using iterative deepening A* with various heurstic improvements. It can navigate sliding tile puzzles, N-ary trees, and octile grid maps.
